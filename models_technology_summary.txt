1. Attendance Agent
   Model: No AI model
   Technology: Pandas, Scikit-learn
   Why: Tabular data analysis ke liye sirf rules aur machine learning algorithms zaruri hain, LLM ki nahi.

2. Day-5 Embedding
   Model: sentence-transformers/all-MiniLM-L6-v2
   Technology: Sentence Transformers
   Why: Semantic similarity aur embedding memory ke liye CPU par fast aur accurate hai.

3. Interview Agent
   Model: RAG components (FAISS, Speech-to-Text, Text-to-Speech)
   Technology: RAG, FAISS, pypdf
   Why: Resume context ko yaad rakhne aur naturally voice mein interact karne ke liye.

4. LangChain RAG Chatbot
   Model: SmolLM2-360M-Instruct (LLM) & BAAI/bge-small-en-v1.5 (Embeddings)
   Technology: LangChain, FAISS
   Why: LangChain RAG architecture ko easy banata hai, aur SmolLM2 fast instruction-following deta hai.

5. Resume Analyzer
   Model: all-MiniLM-L6-v2
   Technology: Cosine Similarity, Sentence Transformers

   Why: Ek full-stack ATS platform banane ke liye jisme fast API aur interactive UI zaroori hai.

7. Buffer Memory
   Model: None

8. PDF Summary

9. Resume Chatbot

17. SmartChunk AI
